# byphantom

The org behind **PHANTOM**, an AI-powered luxury concierge platform for HDFC Bank's top cardholders, operated by R360 / Reward360 Global Services Pvt. Ltd.

Live product: **byphantom.ai** · operator console at `byphantom.ai/concierge`.

## Our 4 repos

| Repo | What it is | Audience |
|---|---|---|
| [hdfc-concierge-platform](https://github.com/byphantom/hdfc-concierge-platform) | The live production app · Next.js 14 · Supabase · Vercel | Engineers, designers, PMs |
| [knowledge-archive](https://github.com/byphantom/knowledge-archive) | Redacted Claude conversations · decision logs · "why we did X" | Onboarders, auditors |
| [phantom-console-legacy](https://github.com/byphantom/phantom-console-legacy) | Frozen historical phantom-console codebase | Reference only |
| [phantom-crawler](https://github.com/byphantom/phantom-crawler) | KB scrapers + raw data dumps | Data engineering |

## Getting started

If you're new to the team, read `ONBOARDING.md` in the repo most relevant to your work. The `knowledge-archive` repo's `ONBOARDING.md` is a good general starting point.
